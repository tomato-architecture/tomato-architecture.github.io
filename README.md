# Tomato Architecture
A pragmatic approach to software development.

## Developer Guide
* Install Mkdocs https://www.mkdocs.org/#installation
    ```shell
    $ python -m venv .venv
    $ source .venv/bin/activate
    $ pip install -r requirements.txt
    ```
    
* Run locally: `mkdocs serve` and open http://127.0.0.1:8000/
